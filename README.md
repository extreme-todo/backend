# Convention

- 함수, 변수는 **camelCase**
- 클래스명 **PascalCase**
- 폴더명, endpoint **snake_case**
- commit message :: 국문
    
    ```jsx
    **Commit 유형 지정**
    (Create) => 새로운 컴포넌트, 기능을 완성
    (Update) => 수정, 추가, 보완 등 개정
    (Delete) => 삭제
    (Install) => 패키지 설치
    (Bug) => 오류 존재
    (Fix) => 오타, 오류 수정
    ex) Create : 회원가입 컴포넌트
    ex) Create : 회원가입 컴포넌트 WIP
    
    **Commit Message 구조**
    type(타입) : title(제목)
    WIP(Work In Progress) : 완성 전에 커밋할 경우 제목 뒤에 붙임
    body(본문, 생략 가능)
    ```
